# 🔮 乾坤 AI 科學面相館 — 智慧命理鑑定系統

本專案為深度學習課程期末報告專案，結合了傳統面相學理、MediaPipe 關鍵點檢測、CNN 卷積神經網路分類，與最新 Gemini LLM 多模態決策引擎，打造出一套兼具學術研究價值與商用質感的面相鑑定系統。

---

## 🚀 快速啟動網頁應用 (Windows 本機)

如果您需要啟動系統的 Web 介面進行測試或 Demo，請按照以下步驟執行：

### 1. 啟動後端 FastAPI 服務
在 Windows 命令提示字元 (CMD) 中執行：
```cmd
cd C:\Users\User\Desktop\深度學習課程\期末報告
set PYTHONIOENCODING=utf-8
python -m uvicorn web_app.main:app --host 0.0.0.0 --port 8000 --reload
```

### 2. 開啟瀏覽器測試
開啟瀏覽器，造訪 👉 **`http://localhost:8000`**。

> 🚨 **重要避坑指南 (防範 404 快取問題)**：
> 如果您在開啟 `http://localhost:8000` 時遇到 CSS / JS 檔案載入失敗或樣式破掉，這是因為瀏覽器快取了舊的專案檔案。
> 1. 請按下 **`Ctrl + Shift + N`** 開啟瀏覽器的**「無痕模式視窗」**。
> 2. 在無痕視窗中造訪 `http://localhost:8000` 即可正常顯示！
> 3. 或者按下 **`Ctrl + Shift + R`** 強制重新整理。

## 🛠️ 常見開發排錯

1. **`[WARN] CNN 模型未載入`**
   如果根目錄下缺少 `models_cnn` 資料夾的權重，系統會自動切換為 **Mock 測試數據** 運作，因此即使沒有模型也可以照常展示網頁！
2. **多模態 API Key**
   本系統預設會自動讀取根目錄 `.env` 檔案內設定的開發者 API Key，如需自訂可在網頁的「進階 LLM 設定」中手動填入。
