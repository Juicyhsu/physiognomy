# 📊 深度學習期末報告 — CNN 對照實驗與 Focal Loss 深入分析報告

本報告針對 8 組 CNN 深度學習面相特徵分類器的對照實驗結果，進行深度的學術級定量與定性分析。

本專案資料集包含 **1,010,315 張五官 ROI 影像**。主要實驗變因如下：
1. **模型架構**：輕量級 `MobileNetV2` (3.4M 參數) 對比 複合縮放的 `EfficientNetB0` (5.3M 參數)。
2. **優化超參數**：學習率（Learning Rate = $1e-4$ vs. $5e-5$）、批次大小（Batch Size = 32 vs. 16）。
3. **損失函數 (核心對照)**：傳統交叉熵 `Categorical Cross-Entropy (CE)` 對比 **`Focal Loss` (調製參數 $\gamma=2.0, \alpha=0.25$)**。

## 2. 深入面向比較與學術定性分析

### 🔍 面向一：類別不平衡度 (Class Imbalance) 與 Focal Loss 的適用邊界（學術上的「有得有失」）

Focal Loss 的數學公式定義為：
$$FL(p_t) = -\alpha_t (1 - p_t)^\gamma \log(p_t)$$
其核心設計是透過調製係數 $(1-p_t)^\gamma$ 來壓低易分類樣本（多數類）的損失貢獻，使梯度更新聚焦於難分類樣本（少數類）。

對照本專案的真實分類結果，我們發現 **Focal Loss 絕非萬靈丹，而是一把雙刃劍，其表現取決於資料集的分布特徵**：

1. **得 ➔ 在「高度不平衡且特徵清晰」的部位（如鼻型）：**
   * **現象與數據**：鼻型分類中，傳統 CE 模型對少數類（挺鼻型）的 Recall 是 **0.0%**（完全無法辨識，直接偏向盲猜佔比 99% 的標準鼻型）。引入 Focal Loss 後（Exp G），挺鼻型 Recall 突破為 **18.6%**，寬鼻型 Recall 飆升至 **47.9% (+42.5%)**，整體 Macro F1-Score **大增了 22.7%**！
   * **學術分析**：這證明了在極端不平衡且少數類特徵相對明確的任務中，Focal Loss 的調製參數能強行拉回梯度方向，逼迫模型學到有用的特徵，起到**「起死回生」**的優化效果。
   
2. **失 ➔ 在「多分類且特徵極度重疊」的部位（如臉型）：**
   * **現象與數據**：臉型 4 分類任務中，引進 Focal Loss 後（Exp G），整體精度下降，且衡量均衡度的 Macro F1-Score 甚至**衰退了 11.0%**，多數類（標準臉型）的 Recall 被拉高到 87.0%，少數類的 Recall 反而全面委縮。
   * **學術分析**：臉型的少數類（圓潤臉）與多數類（標準臉）特徵界線極為模糊。在特徵本身難以區分的前提下，Focal Loss 的重罰機制反而會導致梯度方向劇烈震盪，讓模型陷入局部最優解而產生**過度擬合（Overfitting）**。此時，反而傳統交叉熵（CE）的全局平滑收斂更為穩健。
   
3. **失 ➔ 在「分佈本來就很均衡」的部位（如嘴型）：**
   * **現象與數據**：嘴型的類別分佈極為均勻（約 1.6 : 1.5 : 1）。但在引進 Focal Loss 後，模型的驗證精度在各組中均顯著下滑，尤以 EfficientNet 架構最為嚴重（精度暴跌 -23.8%）。
   * **學術分析**：Focal Loss 的本質是削弱易分類樣本的梯度。在均衡資料集中，各類別樣本的預測難度相近，Focal Loss 會不斷且無差別地削弱正常樣本的權重，反而破壞了損失函數的平滑收斂軌跡，導致訓練震盪。

**💡 學術總結論（Ensemble 集成的核心論據）：**
機器學習任務中不應盲目對所有部位統一使用同一個損失函數。本專案的最終優化策略是：**「對眼、鼻、眉（不平衡任務）使用 Focal Loss，對嘴、臉（均衡或特徵模糊任務）沿用傳統交叉熵（CE）」**。這種「因地制宜」的集成模型（Ensemble），在學術上比單一的「Focal Loss 全局大勝」論述更具科學深度與說服力。


### 🔍 面向二：優化超參數（學習率、Batch Size）的協同作用

1. **小學習率 (lowLR) 的影響**：
   * **數據**：在實驗 G 中，**鼻型分類精度從 53.0% 提高至 58.4% (+5.4%)**，取得全場最佳鼻型精度。
   * **分析**：Focal Loss 的損失曲面比傳統 CE 更為崎嶇且敏感。使用較小的學習率 ($5e-5$) 能有效避免模型在梯度回傳時產生震盪，有助於在非極端不平衡的任務（如鼻型 2.6:1.4:1）中平滑收斂至全局最優解。
2. **小批次量 (Batch Size = 16) 的影響**：
   * **數據**：實驗 H（小批次 + Focal Loss）在**眼型 (72.8%)** 與 **眉型 (64.4%)** 上均取得了全場最高的驗證精度。
   * **分析**：Batch Size 減半意味著每次梯度更新時使用的隨機樣本數減少，這會引入額外的隨機梯度噪聲（Stochastic Noise），在數學上具有**正則化（Regularization）**的效果。眼型與眉型屬於面相中局部、微小且容易過擬合的視覺特徵，小 Batch Size 帶來的情境噪聲與 Focal Loss 聚焦難分樣本的特性結合，有效提升了特徵泛化能力。

### 🔍 面向三：模型架構與 Focal Loss 的相容性

* **MobileNetV2 (實驗 A, E, C, G, D, H)**：整體表現相對平穩，在 Focal Loss 引入後展現了穩定收斂。
* **EfficientNetB0 (實驗 B, F)**：引入 Focal Loss 後，在均衡任務（嘴型）中準確率從 63.2% 斷崖式下跌至 39.5%（衰退 -23.8%）。
* **分析**：EfficientNetB0 內部使用了 Swish 激活函數與 MBConv 殘差塊，其特徵表徵能力較強。當搭配 Focal Loss 處理均衡類別時，模型內部複雜的神經元激活更容易受到 Focal Loss 損失調製的干擾，產生梯度梯度爆炸或消失，因此對資料均衡性更為敏感。
